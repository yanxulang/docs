## 第一次学习言序

建议按以下顺序阅读：

1. [基础语法与类型](/language/basics/)：理解值、绑定、类型标注和表达式。
2. [控制流](/language/control-flow/)：使用条件与循环组织执行顺序。
3. [函数与闭包](/language/functions/)：提取可复用逻辑，并理解词法作用域。
4. [容器与迭代](/language/collections/)：处理列、典、下标和数据遍历。
5. [类与实例](/language/objects/)：在需要状态与行为时建立对象抽象。
6. [模块](/language/modules/)：把多个文卷组织成可复用单元。
7. [错误处理](/language/errors/)：为失败路径补充诊断、捕获和恢复逻辑。

完成前四步即可编写小型数据处理程序；进入模块和错误处理后，再开始组织需要长期维护的项目。

## 工程实践路线

完成语言基础后，阅读[VS Code 扩展](/tooling/vscode/)和[从源码构建](/tooling/source-build/)，逐步建立适合项目的编辑、测试与执行流程。

准备修改词法器、解析器或解释器时，再进入[运行时架构](/project/architecture/)；[长期路线图](/project/roadmap/)用于理解版本边界和后续方向，不作为初学者的前置材料。
