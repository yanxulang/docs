---
title: 配置编辑器
description: 安装官方 VS Code 扩展，并确认高亮、格式化、诊断与调试入口。
---

在 VS Code 扩展市场搜索 **言序**，或从[官方扩展仓库](https://github.com/YanXuLang/vscode-extension)安装当前发布版 1.4.0。扩展识别 `.yx` 文件和 `yanxu`、`yx`、`言序`三种代码块语言标识。

打开一个 `.yx` 文卷后，确认以下能力可用：

- 关键字、类型、声明、调用、文字、数字与注释使用不同语义颜色；
- 保存或“格式化文档”调用 `yanxu 格`；
- 问题面板显示词法、语法与类型诊断；
- 定义、引用、重命名、悬停、补全和文档符号来自 `yanxu 语言服务`；
- 调试配置通过 `yanxu 调试服务`连接 DAP。

扩展默认从 `PATH` 查找 `yanxu`。如果终端能执行 `yanxu --version`，但扩展仍找不到核心，请在 VS Code 中重载窗口，并确认图形应用继承了更新后的环境变量。

编辑器只是核心工具的入口：CI 和命令行仍应直接运行检查、格式化与测试。完整设置和调试配置见[VS Code 扩展](/tooling/vscode/)。
