---
title: 第一份程序
description: 创建、理解并运行第一份言序源码文件。
---

新建 UTF-8 编码的 `初见.yx`：

```yanxu
令 名字：文 为 「言序」；
言 「你好，」 加 名字；
```

在文件所在目录执行：

```sh
yanxu 初见.yx
```

终端将输出 `你好，言序`。

第一行以`令`声明名为`名字`的变量，`：文`要求它保存文字，`为`之后是初始值。第二行以`言`输出两段文字相加后的结果。言序源码推荐使用 `.yx` 扩展名，语句以`；`结束。
