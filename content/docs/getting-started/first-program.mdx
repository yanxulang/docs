---
title: 第一份程序
description: 创建、理解、检查并运行第一份完整言序源码文卷。
---

新建 UTF-8 编码的 `初见.yx`：

```yanxu
令 名字：文 为 「言序」；

言 （「你好，」 加 名字）；
```

先检查，再运行：

```sh
yanxu check 初见.yx
yanxu 初见.yx
```

终端应输出：

```text
你好，言序
```

第一行用`令`创建可重新赋值的`名字`，`：文`把它约束为文字，`为`之后给出初值。第二行用`加`连接两段文字，再由`言`输出。言序源码推荐使用 `.yx` 扩展名，完整语句以`；`结束。

`yanxu check`只做词法、语法、语义与类型检查，不执行用户代码。要把规范格式写回文卷，可运行：

```sh
yanxu fmt --write 初见.yx
```

本页程序由 CI 使用言序 1.1.8 实际检查、格式化和执行。下一步把单一文卷扩展成[第一个项目](/getting-started/first-project/)。
