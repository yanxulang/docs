---
title: 安装
description: 在 macOS、Linux 或 Windows 上安装并验证言序。
---

## macOS 与 Linux

```sh
curl -fsSL https://get.yanxu.dev | sh
```

## Windows PowerShell

```powershell
irm https://get.yanxu.dev/windows | iex
```

## 验证安装

安装完成后查看版本：

```sh
yanxu --version
```

项目初始化、依赖管理和打包由官方[言包](/tooling/package-manager/)完成。安装言序后，再安装言包：

```sh
curl -fsSL https://get.yanxu.dev/yanbao | sh
```

言包 0.4.0 完全由言序编写，复用`PATH`中的言序 1.1.6 或更高版本；不需要 Rust，也不会安装或覆盖另一份运行时。

如果你希望参与语言实现，或安装器不适用于当前环境，请改用[从源码构建](/tooling/source-build/)。
