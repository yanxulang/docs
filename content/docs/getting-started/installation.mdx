---
title: 安装
description: 在 macOS、Linux 或 Windows 上安装并验证言序。
---

## macOS 与 Linux

```sh
curl -fsSL https://get.yanxu.dev | sh
```

## Windows PowerShell

```powershell
irm https://get.yanxu.dev/windows | iex
```

## 验证安装

安装完成后查看版本：

```sh
yanxu --version
```

项目初始化、依赖管理和打包由独立的[言包](/tooling/package-manager/)完成。言包 0.2.1 的命令实现完全由言序编写，正式 Release 为每个平台附带专用的言序 1.1.6 运行时和一键安装脚本，因此普通用户不需要先安装言序或 Rust。只有从言包源码仓库直接开发时，才需要自行准备兼容的言序核心。

如果你希望参与语言实现，或安装器不适用于当前环境，请改用[从源码构建](/tooling/source-build/)。
