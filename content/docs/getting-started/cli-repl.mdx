## 常用命令

| 命令 | 用途 |
| --- | --- |
| `yanxu 文卷.yx` | 执行源码文件 |
| `yanxu` | 进入多行交互环境 |
| `yanxu --version` | 显示版本 |
| `yanxu --help` | 显示命令说明 |

## 交互环境

不带文件运行`yanxu`即可进入 REPL。它会保留先前定义，类、函数、判断和循环可以跨多行输入；`终`闭合后统一执行。表达式语句显示为`⇒ 值`。

| 命令 | 用途 |
| --- | --- |
| `:助` / `:help` | 查看帮助 |
| `:退` / `:quit` | 退出 |
| `:清` / `:clear` | 重建解释器并清空上下文 |
| `:载 路径.yx` | 在当前上下文执行文卷 |

REPL 适合快速试验；需要复用或协作的代码应保存为 `.yx` 文卷，并通过[模块](/language/modules/)组织。
