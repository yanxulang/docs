```yanxu
公 法 平方（本数：数）：数 则
    归 本数 乘 本数；
终
```

在另一卷中引用：

```yanxu
引「模块/数术.yx」为 数术；
言 数术.平方（9）；
```

也可写作`引「模块/数术.yx」作 数术；`。相对路径以包含导入语句的文卷目录为基准，不受启动命令所在目录影响。

每个模块在独立词法环境中执行。顶层声明默认私有，只有以`公`修饰的变量、常量、法和类会进入模块命名空间。公开变量保持活绑定：模块内更改后，外部下次读取会看到新值。

模块只执行一次，后续引用复用缓存；成员不能由外部改写，循环引用会报错。

## 包与标准模块

`yanxu 包 .`会向上寻找`言序.toml`：

```toml
[包]
名 = "算书"
格式 = 1
版 = "1.0.0"
入口 = "src/主.yx"

[依赖]
工具 = "../tools"
```

依赖也可使用完整表描述路径、Git 或中央索引来源。`yanxu 包 锁`生成/验证`言序.lock`，固定语义版本、Git 精确修订、入口和 SHA-256；`yanxu 包 更新`才会显式重锁，`--离线`只复用已经校验的缓存。

`引「包:工具」为 工具；`按锁文件从包根目录解析依赖。模块按确定顺序初始化；循环诊断会显示完整的`甲 → 乙 → 甲`链。

1.0 稳定集合包括`文字`、`数学`、`时间`、`文件`、`JSON`、`网络`、`测试`、`路径`、`环境`、`哈希`、`编码`、`统计`与`CSV`；其后以兼容方式加入`随机`、`标识`、`模板`和`校验`，目前共 17 个。例如`引「标准:数学」为 数学；`可访问`圆周率`、`绝对值`、`平方根`与`幂`。完整 API 见[标准库](/language/standard-library/)，模块运行机制见[运行时架构](/project/architecture/#模块)。
