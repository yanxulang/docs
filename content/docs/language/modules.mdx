```yanxu
引「模块/数术.yx」为 数术；
言 数术.平方（9）；
```

也可写作`引「模块/数术.yx」作 数术；`。相对路径以包含导入语句的文卷目录为基准，不受启动命令所在目录影响。

每个模块在独立词法环境中执行，并以该环境的顶层绑定形成命名空间。模块只执行一次，后续导入复用缓存。模块成员不能由外部改写，循环导入会报错。

当前版本导出模块的全部顶层绑定；未来版本将引入`公`关键字并默认隐藏未公开成员。完整的模块运行机制见[运行时架构](/project/architecture/#模块)。
