## 条件分支

```yanxu
若 年岁 不小于 18 且 有凭据 则
    言 「可入」；
否则
    言 「止步」；
终
```

`若`在条件成立时执行第一段正文，可选的`否则`在条件不成立时执行。`假`与`空`视为不成立，其余值视为成立。

## 条件循环

```yanxu
令 次数：数 为 0；

当 次数 小于 3 则
    言 次数；
    置 次数 为 次数 加 1；
终
```

`当`会在每轮开始前重新求值条件。需要遍历列、文字或典时，优先使用[统一迭代](/language/collections/#统一迭代)，以避免手动管理下标。

函数中的`归`会立即结束当前函数；错误处理中的`抛`会沿调用路径退出，直到被`救`捕获。
