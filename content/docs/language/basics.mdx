## 值与绑定

```yanxu
令 数目：数 为 42；
令 比例：数 为 3.14；
令 题名：文 为 「天地玄黄」；
令 可行：理 为 真；
令 未有：空 为 空；
令 灵活：任意 为 42；
定 名录：列 为【「甲」，「乙」】；

置 数目 为 数目 加 1；
```

`令`创建可以重新赋值的变量，`定`创建不可重新赋值的绑定。`定`只约束绑定本身，不会递归冻结它所指向的列或典。

类型标注可以省略；省略后变量可接纳任意类型。带标注变量在初次声明和每次`置`时都会检查。数值目前使用十进制阿拉伯数字，文字可写成`「文字」`、`“文字”`或`"文字"`。

## 内置类型

| 类型 | 含义 |
| --- | --- |
| `数` | 数值 |
| `文` | 文字 |
| `理` | 真或假 |
| `空` | 空值 |
| `法` | 用户函数或内置函数 |
| `类` | 类本身 |
| `对象` | 任意类的实例 |
| `模块` | 模块命名空间 |
| `列` | 有序可变序列 |
| `典` | 保持插入顺序的映射 |
| `误` | 可捕获的结构化运行错误 |
| `任意` | 不限制值类型 |
| 类名 | 指定类的实例 |

## 运算与优先级

| 层级（由高到低） | 运算符 |
| --- | --- |
| 调用、成员、下标 | `函数（参数）`、`对象.成员`、`容器【下标】` |
| 一元 | `非`、`!`、`-` |
| 乘除 | `乘`、`除`、`*`、`/`、`×`、`÷` |
| 加减 | `加`、`减`、`+`、`-` |
| 比较 | `大于`、`不小于`、`小于`、`不大于`、`>`、`>=`、`<`、`<=` |
| 相等 | `等于`、`不等于`、`==`、`!=` |
| 逻辑且 | `且` |
| 逻辑或 | `或` |

`且`和`或`会短路求值。`加`既可用于两个数相加，也可用于两段文字相接。括号可以显式改变求值顺序。
