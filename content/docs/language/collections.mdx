## 列与典

```yanxu
定 名录：列 为【「甲」，「乙」】；
追加（名录，「丙」）；
置 名录【1】为「仲」；
言 名录【0】；
言 弹出（名录）；

令 次序：典 为 {「甲」：1，「仲」：2}；
置 次序【「丙」】为 3；
言 次序【「仲」】；
言 有键（次序，「丙」）；
```

列可用`【】`或`[]`。典使用`{键：值}`，键仅可为数、文、理或空；重复键保留原位置并更新值。

列下标和文字下标须为从零开始的非负整数。读取不存在的键或越界下标会产生可捕获错误。`长度`适用于文字、列和典；`追加`与`弹出`会原地修改列。

## 统一迭代

```yanxu
逐 项：文 于 名录 则
    言 项；
终

逐 键 于 次序 则
    言 次序【键】；
终
```

逐列得到元素，逐文字得到单字符文字，逐典得到键。每次迭代的变量都是新的不可改写绑定，闭包可安全捕获当次值。

迭代期间发生的下标错误或用户代码错误，可以使用[结构化错误处理](/language/errors/)捕获。
