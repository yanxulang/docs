## VS Code 扩展

官方扩展提供：

- `.yx` 文卷语法高亮；
- 中文引号与括号自动配对；
- 变量、判断、循环、函数、类和模块代码片段；
- 一键运行当前文卷；
- 打开言序 REPL 与在线文档的命令。

从 [VS Code 扩展发行页](https://github.com/YanXuLang/vscode/releases)下载 `.vsix`，在 VS Code 扩展面板中选择“从 VSIX 安装”。

## 从源码构建

```sh
git clone https://github.com/YanXuLang/language.git
cd language
cargo test --all-targets --locked
cargo install --path .
```
