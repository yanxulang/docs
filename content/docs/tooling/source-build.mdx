## 构建语言核心

需要安装稳定版 Rust 工具链和 Git：

```sh
git clone https://github.com/YanXuLang/yanxu.git
cd yanxu
cargo test --all-targets --locked
cargo install --path .
```

安装后运行`yanxu --version`确认当前终端可以找到生成的可执行文件。
