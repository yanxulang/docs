---
title: VS Code 扩展
description: 安装并使用言序官方 VS Code 扩展。
---

官方扩展提供：

- `.yx` 文卷语法高亮；
- 中文引号与括号自动配对；
- 变量、判断、循环、函数、类和模块代码片段；
- 一键运行当前文卷；
- 自动启动 1.0 LSP，显示诊断并提供补全、跳转、引用、重命名、悬停和文档符号；
- 使用言序格式化器设置全文格式；
- 命令面板运行静态检查、独立字节码 VM、执行踪迹与工作区测试；
- 按格式 2 根依赖别名和显式导出索引`包:`模块，并隔离未声明的传递依赖；
- 一键编译工作区 YXB 应用，离线快照覆盖 25 个标准模块；
- 使用 VS Code“运行和调试”设置断点、继续/单步并检查调用栈与变量；
- 打开言序 REPL 与在线文档的命令。

从 [VS Code 扩展发行页](https://github.com/YanXuLang/vscode-extension/releases)下载 1.3.1 `.vsix`，在 VS Code 扩展面板中选择“从 VSIX 安装”。扩展面向言序 1.1.5，默认从`PATH`寻找`yanxu`，也可使用`yanxu.executablePath`指定可执行文件。扩展本身不要求安装言包。

扩展问题与功能建议请提交到 [YanXuLang/vscode-extension](https://github.com/YanXuLang/vscode-extension/issues)。
