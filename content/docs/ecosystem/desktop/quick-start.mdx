---
title: 快速开始
description: 使用言序 1.1.9、言包 0.5.0 与言界 0.1.1 复现已验证的预览组合。
---

<Callout type="warning" title="不要与当前稳定工具链混用">
  本页复现言界 0.1.1 发布时的已验证组合：言序 1.1.9 与言包 0.5.0。言序 1.1.20 会在言界源码的可空目标变量处报告类型错误，言包 0.6.1 也不能运行在 1.1.9 上。使用当前稳定工具链时，请改走[言窗 1.0](/ecosystem/desktop/gui/)。
</Callout>

先确认`yanxu --version`为`1.1.9`，`yanbao --version`为`0.5.0`。创建普通言序项目后，在`言序.toml`中加入言界：

```toml
[包]
格式 = 2
名称 = "我的窗口"
版本 = "0.1.0"
言序 = ">=1.1.9"
入口 = "src/主.yx"

[依赖]
言界 = { 包 = "yanxu-ui", git = "https://github.com/yanxulang/yanxu-ui.git", 修订 = "v0.1.1", 版 = "^0.1" }

[权限]
图形界面 = true
原生扩展 = true
剪贴板 = true
文件对话框 = true
```

把以下代码保存为`src/主.yx`：

```yanxu
引「包:言界」为 界面；

定 应用 为 界面.应用（「你好」）；
定 窗口 为 应用.窗口（{「标题」：「你好，言界」，「宽」：640，「高」：420}）；
定 布局 为 窗口.列（{「内边距」：24，「间距」：12}）；
定 输入 为 布局.输入框（{「占位」：「请输入中文」}）；
定 提示 为 布局.文字（「尚未点击」）；
定 按钮 为 布局.按钮（「问候」）；

法 点击（所事件） 则
    提示.内容（「你好，」 加 输入.取内容（））；
终

法 关闭（所事件） 则
    应用.退出（）；
终

按钮.点击（点击）；
窗口.关闭时（关闭）；
窗口.显示（）；
应用.运行（）；
```

安装、检查并运行：

```sh
yanbao update
yanbao check
yanbao run
```

第一次更新会为当前目标下载锁定的言台原生后端。运行后可在输入框测试中文 IME，再点击“问候”。若只需要显示文字，可继续阅读[创建第一个窗口](/ecosystem/desktop/first-window/)；发布独立应用见[打包与发布](/ecosystem/desktop/packaging/)。
