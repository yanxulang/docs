---
title: 言据配置
description: 用言据编写可读、可规范化且带版本的言界主题配置。
---

言界把言据作为主题、样式和调试交换的首选文本格式。当前主题协议主版本为`1`：

```yanju
据【
  「版本」：1，
  「名称」：「海蓝」，
  「基于」：「浅色」，
  「颜色」：据【「主色」：列【20，112，210，255】】，
  「度量」：据【「圆角」：10】，
  「样式」：据【
    「按钮」：据【
      「背景」：列【20，112，210，255】，
      「前景」：列【255，255，255，255】
    】
  】
】
```

解析并应用：

```yanxu
引「包:言界」为 界面；
引「包:言界/配置」为 配置；

定 数据 为 配置.解析配置（主题原文，「言据」）；
定 主题 为 配置.规范主题（数据）；
定 应用 为 界面.应用（「言据主题」）；
应用.主题配置（主题）；
```

`规范主题`检查版本、必需结构、颜色与度量范围，并补齐稳定默认值。不同主版本返回`UI_CONFIG_MAJOR`；未知可选字段会保留，以便新旧工具往返。`输出配置（数据，「言据」）`可生成稳定文本，适合版本控制和调试快照。

言据文本只在启动、交换或调试时解析。事件批次和逐帧绘制使用原生值与二进制协议，不会把言据放进热路径。
