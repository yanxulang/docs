---
title: 样式与主题
description: 使用内置浅深主题、控件状态样式和可继承自定义主题。
---

应用默认使用浅色主题，也可切换内置主题：

```yanxu
应用.主题（「深色」）；
应用.主题（「浅色」）；
```

控件最终样式按以下顺序合成：主题基础样式、父级继承、`悬停`、`按下`、`焦点`、`禁用`状态，最后是控件实例的`样式`。循环继承会返回`UI_STYLE_CYCLE`，而不是递归到栈溢出。

```yanxu
定 主要按钮 为 布局.按钮（「继续」）；
主要按钮.样式（{
    「背景」：【20，112，210，255】，
    「前景」：【255，255，255，255】，
    「圆角」：10
}）；
```

颜色使用`【红，绿，蓝，透明度】`的 0–255 整数列；几何度量使用逻辑像素。应用级设计系统应写成[言据配置](/ecosystem/desktop/yanju-config/)，再调用`应用.主题配置（配置.规范主题（数据））`。JSON 可作为外部工具兼容输入，但不是唯一格式。

系统主题变化会成为应用事件。是否自动跟随由应用决定；切换主题后，言界标记受影响控件的绘制脏区并生成新帧。
