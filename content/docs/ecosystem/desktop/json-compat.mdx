---
title: JSON 兼容
description: 为外部工具和历史程序保留与言据等价的 JSON 配置入口。
---

JSON 不是言界唯一数据格式，但外部工具可以使用与言据主题相同的数据模型：

```json
{
  "版本": 1,
  "名称": "海蓝",
  "基于": "浅色",
  "颜色": { "主色": [20, 112, 210, 255] },
  "度量": { "圆角": 10 },
  "样式": {
    "按钮": { "背景": [20, 112, 210, 255], "前景": [255, 255, 255, 255] }
  }
}
```

```yanxu
引「包:言界/配置」为 配置；
定 数据 为 配置.解析配置（JSON原文，「JSON」）；
应用.主题配置（配置.规范主题（数据））；
```

格式参数`自动`会先尝试言据，再尝试 JSON；需要确定错误位置和协议来源时应显式写`言据`或`JSON`。`输出配置`以及主题对象的`言据`／`JSON`方法提供稳定往返。

两种文本入口经过同一规范化和版本检查，未知可选字段策略相同。JSON 兼容层不会改变言包现有的版本化 JSON 工程协议，也不会进入事件或绘制热路径。
