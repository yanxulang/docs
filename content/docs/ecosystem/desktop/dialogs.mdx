---
title: 文件对话框
description: 打开文件、保存文件和选择目录，并正确处理取消结果。
---

文件对话框需要独立权限：

```toml
[权限]
文件对话框 = true
```

言界公开`打开文件`、`打开多个文件`、`保存文件`、`选择目录`和`选择多个目录`。配置可包含`标题`、`目录`、`文件名`与过滤器：

```yanxu
定 路径 为 界面.打开文件（{
    「标题」：「打开言序源码」，
    「过滤器」：【
        {「名称」：「言序源码」，「扩展」：【「yx」】}，
        {「名称」：「全部文件」，「扩展」：【「*」】}
    】
}）；
```

用户取消单选时返回空，多选成功时返回路径列。路径使用宿主可表示的 Unicode 文本。选择路径不等于授予任意文件读写能力；应用仍应按言序自身的文件权限和错误模型打开文件。

对话框由系统服务实现，但业务确认框、菜单和弹出层属于言界控件，不应塞入言台后端。
