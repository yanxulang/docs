言界输入框使用 Unicode 字符位置维护文档、光标和锚点，并在与言台文字服务交互时显式转换 UTF-8 字节位置。它不把字节、Unicode 标量和可见字形混为一谈。

```yanxu
定 单行 为 布局.输入框（{「占位」：「姓名」}）；
定 多行 为 布局.多行输入框（{
    「内容」：「中文 English 😀 𠀀\n」，
    「占位」：「支持中文输入法」，
    「首选高」：180
}）；

法 内容变化（所事件） 则
    回显.内容（「当前内容：」 加 单行.取内容（））；
终
单行.变化时（内容变化）；
```

已实现插入、退格、向前删除、正反向选区、全选、复制、剪切、粘贴、256 项撤销历史、左右上下、Home、End、Shift 扩展、鼠标定位、拖动选区和多行滚动。

## 中文组合生命周期

```text
IME组合开始
  → IME组合更新（预编辑文字与可选选区，可重复）
  → IME组合提交（正式写入文档与撤销历史）
或→ IME组合取消（不改变正文）
```

输入框获得焦点后启用文本输入，并把当前光标的逻辑像素矩形传给系统，以定位拼音、五笔等候选窗。失焦时取消组合并停止输入法。组合更新替换上一段预编辑，不能追加；提交文字也不能再次从按键事件插入。

CI 验证中文预编辑`言序`的事件映射、整形、索引和字形重编码。真正的候选窗仍依赖已安装输入法和用户桌面会话，因此发布前应在三套目标系统进行人工输入冒烟。
