---
title: 言版：语义版本
description: 严格解析 SemVer 2.0.0，并比较、匹配范围与选择最高兼容版本。
---

言版（`yanxu-semver`）实现 SemVer 2.0.0 的严格版本解析和优先级规则，并提供工程中常用的版本范围匹配。

## 安装

```sh
yanbao add semver --package 言版 --version "^0.1"
```

言版是无权限、无传递依赖的纯计算库。

## 解析与比较

```yanxu
引「包:言版」为 言版；

定 版本 为 言版.解析（「1.4.0-beta.2+build.9」）；
言 版本；
言 言版.规范化（「1.4.0-beta.2+build.9」）；
言 言版.比较（「1.4.0-beta.2」，「1.4.0」）； # -1
```

`比较` 返回 `-1`、`0` 或 `1`。构建标识不参与优先级；预发布标识按 SemVer 数字与文字规则逐段比较。

## 范围匹配

```yanxu
言 言版.满足（「1.4.0」，「^1.2.0」）；
言 言版.满足（「2.0.0-beta.1」，「>=2.0.0-beta.1 <2.0.0」）；
言 言版.满足（「3.1.5」，「2.x || 3.1.x」）；
```

支持：

- `=`、`>`、`>=`、`<`、`<=`；
- 空格连接的交集，例如 `>=1.2 <2.0`；
- `||` 连接的并集；
- 尖帽范围 `^`；
- 波浪范围 `~`；
- 部分版本与 `x`、`X`、`*` 通配。

范围语法刻意保持确定，不接受连字符范围和括号。

## 选择兼容版本

```yanxu
定 可用 为 【「1.2.0」，「1.7.3」，「2.0.0-beta.1」，「2.0.0」】；

言 言版.排序版本（可用）；
言 言版.最大满足（可用，「^1.2.0」）； # 1.7.3
```

`排序版本` 返回新列，不改写输入。`最大满足` 没有候选时返回空，因此调用方应显式处理未命中分支。

## 公共接口

| 接口 | 说明 |
| --- | --- |
| `解析` | 严格解析为结构化典 |
| `是否合法` | 不抛错的合法性判断 |
| `规范化` | 返回标准版本文字 |
| `比较` | 比较两个版本优先级 |
| `满足` | 检查候选是否匹配范围 |
| `排序版本` | 按优先级升序返回新列 |
| `最大满足` | 选择范围内最高版本或空 |

解析与范围错误使用 `YANBAN_` 前缀。接收不可信范围文字时，可先调用 `是否合法` 检查单个版本；范围本身仍应在应用边界捕获并转换为用户可读错误。
