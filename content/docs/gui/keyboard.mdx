---
title: 键盘与快捷键
description: 区分物理键、逻辑键、文字输入与跨平台主修饰键。
---

键盘事件同时保留`物理键`、`逻辑键`、可选`文本`、键盘`位置`、`修饰键`、`重复`和`合成`状态。物理键描述键位（如`KeyA`），逻辑键描述当前布局含义（如`a`或`Enter`）。

应用快捷键使用标准化签名；`主`在 macOS 对应 Command，在 Windows 和 Linux 对应 Control：

```yanxu
法 保存（所事件） 则
    状态.内容（「已保存」）；
终

窗口.快捷（「主+s」，保存）；
```

按钮在获得焦点时响应 Enter 和空格，输入控件实现方向键、Home、End、Shift 扩展选区以及平台常用的复制、剪切、粘贴、全选、撤销和重做组合。

不要根据按键名称推导要插入的字符：不同键盘布局、死键和输入法会使该推导出错。编辑器只从`文本输入`或`IME组合提交`写入正文；按键事件只处理命令、导航和快捷键。`重复`为真时可重复导航或删除，但应用级一次性操作通常应忽略重复。
