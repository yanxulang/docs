## 六个正式目标

```text
x86_64-pc-windows-msvc
aarch64-pc-windows-msvc
x86_64-apple-darwin
aarch64-apple-darwin
x86_64-unknown-linux-gnu
aarch64-unknown-linux-gnu
```

每个目标都会核对动态库实际架构、包清单摘要、言序 1.1.7 类型检查、ABI 集成、API 漂移和 Release 构建。真实窗口自动退出在六个执行器运行；Linux 使用 Xvfb 验证 X11 路径，Wayland 与实际输入法会话需另做桌面冒烟。

运行时应调用言台`能力查询（）`，根据字段判断可选能力；不要只根据操作系统名称猜测。
