言序是一门由 Rust 实现的现代中文编程语言。你可以使用中文关键字、中文标识符和全角标点，同时获得类型、容器、模块与结构化错误等工程能力。

## 使用言包管理项目

言包（`yanbao`）是言序项目的默认入口。创建项目、添加依赖和运行检查均通过一套稳定命令完成：

```sh
yanbao init 我的项目 --name 示例
cd 我的项目
yanbao add http
yanbao check
yanbao test
```

官方 GitHub 包可省略`yanxulang/`组织名和`yanxu-`前缀：`yanbao add http`等价于添加`yanxulang/yanxu-http`。完整安装与来源规则见[言包工程工具](/tooling/package-manager/)。

## 建议阅读顺序

第一次接触言序时，先完成[安装](/getting-started/installation/)和[第一份程序](/getting-started/first-program/)，再沿[学习路线](/getting-started/learning-path/)逐步进入语言指南。需要保存应用配置或交换结构化数据时，继续阅读[言据](/yanju/)；开始构建真实应用时，可从[第三方库](/libraries/)选择验证、日志、数据访问、日期时间和测试能力。已经熟悉言序时，可以直接把语言指南、言据、第三方库和工具链页面作为速查手册使用。

如果你准备参与实现或工具开发，请继续阅读[从源码构建](/tooling/source-build/)和[运行时架构](/project/architecture/)。

## 当前版本

文档对应言序核心`1.1.8`、言包`0.5.0`、言据`1.1.2`、言台`0.1.0`与言界`0.1.0`。本版在 ABI v1 和旧 YXB 兼容的基础上提供原生 ABI v2、持久回调、有界宿主事件队列、代际父子资源、独立 GUI 权限、可携带锁定原生后端的 YXB，以及 Windows、macOS、Linux 桌面 Bundle；1.1.8 还修复 Windows 图形回调路径的运行时栈不足。现有`yanxu-gui`（言窗）继续提供成熟的立即模式路线；新路线由`yanxu-ui`（言界）以言序实现保留模式控件，并经`yanxu-platform`（言台）访问统一平台原语。核心仍不把 GUI 或 Web 框架塞进标准库。开始桌面开发前请阅读[图形界面概览](/gui/)、[选择 GUI 路线](/gui/choosing/)、[1.1.8 迁移指南](/tooling/migration-1.1.8/)与[兼容政策](/project/compatibility/)。

需要立即获取发行版时，可前往[下载中心](https://yanxu.dev/download/)；可运行示例位于[语言核心仓库](https://github.com/YanXuLang/yanxu/tree/main/examples)。
