言序是一门由 Rust 实现的现代中文编程语言。你可以使用中文关键字、中文标识符和全角标点，同时获得类型、容器、模块与结构化错误等工程能力。

## 建议阅读顺序

第一次接触言序时，先完成[安装](/getting-started/installation/)和[第一份程序](/getting-started/first-program/)，再沿[学习路线](/getting-started/learning-path/)逐步进入语言指南。已经熟悉言序时，可以直接把语言指南、标准库和工具链页面作为速查手册使用。

如果你准备参与实现或工具开发，请继续阅读[从源码构建](/tooling/source-build/)和[运行时架构](/project/architecture/)。

## 当前版本

文档对应言序 `1.1.4`。语言规范、1.x 源码兼容政策、不可变字节串、二进制文件/HTTP/TCP/UDP、安全随机与 HMAC、生产级 HTTPS、安全受限的套接字、格式 1 包与锁、双执行器、23 个标准模块、沙箱嵌入、语义 LSP、DAP、测试、API 文档与迁移工具均已完成验收。1.1.4 是网络应用的底层能力版本，不包含 HTML、HTTP 服务端或 Web 框架。升级和扩展语言前请阅读[兼容政策](/project/compatibility/)与[长期路线图](/project/roadmap/)。

需要立即获取发行版时，可前往[下载中心](https://yanxu.dev/download/)；可运行示例位于[语言核心仓库](https://github.com/YanXuLang/yanxu/tree/main/examples)。
