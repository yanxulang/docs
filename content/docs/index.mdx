言序是一门由 Rust 实现的现代中文编程语言。你可以使用中文关键字、中文标识符和全角标点，同时获得类型、容器、模块与结构化错误等工程能力。

## 使用言包管理项目

言包（`yanbao`）是言序项目的默认入口。创建项目、添加依赖和运行检查均通过一套稳定命令完成：

```sh
yanbao init 我的项目 --name 示例
cd 我的项目
yanbao add http
yanbao check
yanbao test
```

官方 GitHub 包可省略`yanxulang/`组织名和`yanxu-`前缀：`yanbao add http`等价于添加`yanxulang/yanxu-http`。完整安装与来源规则见[言包工程工具](/tooling/package-manager/)。

## 建议阅读顺序

第一次接触言序时，先完成[安装](/getting-started/installation/)和[第一份程序](/getting-started/first-program/)，再沿[学习路线](/getting-started/learning-path/)逐步进入语言指南。需要保存应用配置或交换结构化数据时，继续阅读[言据](/yanju/)；已经熟悉言序时，可以直接把语言指南、言据和工具链页面作为速查手册使用。

如果你准备参与实现或工具开发，请继续阅读[从源码构建](/tooling/source-build/)和[运行时架构](/project/architecture/)。

## 当前版本

文档对应言序核心`1.1.6`与言包`0.4.0`。格式 2 清单与完整锁图、源码无关 YXB、独立应用、原生 ABI v1 和 25 个标准模块保持兼容。言包完全由言序编写，复用系统中已安装的言序 1.1.6 或更高版本，并以 GitHub 作为默认包来源。核心运行时不内置 Web 框架；Web 能力由独立包提供。升级前请阅读[1.1.6 迁移指南](/tooling/migration-1.1.6/)、[兼容政策](/project/compatibility/)与[长期路线图](/project/roadmap/)。

需要立即获取发行版时，可前往[下载中心](https://yanxu.dev/download/)；可运行示例位于[语言核心仓库](https://github.com/YanXuLang/yanxu/tree/main/examples)。
