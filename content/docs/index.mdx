言序是一门由 Rust 实现的现代中文编程语言。中文关键字、中文标识符与全角标点直接进入语法；类型、模块、结构化错误、树解释器、字节码 VM 和工程工具共同约束程序边界。

<Callout type="info" title="当前稳定基线">
  本站对应言序核心 1.1.20、言包 0.6.1、言据 1.2.0 与 VS Code 扩展 1.4.0。1.1.18 与 1.1.19 没有形成稳定 Release；从 1.1.17 应直接升级到 1.1.20。页面会区分语言核心、标准库、官方工具和独立生态包，不把规划中的能力写成现状。
</Callout>

## 建议阅读顺序

1. 完成[安装](/getting-started/installation/)、[编辑器设置](/getting-started/editor/)和[第一份程序](/getting-started/first-program/)。
2. 创建[第一个项目](/getting-started/first-project/)，形成[日常反馈循环](/getting-started/daily-workflow/)。
3. 沿[语言学习路线](/language/)掌握可复用程序所需的核心概念。
4. 在需要时进入[项目工程](/projects/)、[标准库](/standard-library/)或[实践指南](/guides/)。

语言核心不内置 Web 框架或桌面控件；这些能力在独立生态包中按自己的版本与兼容政策演进。若你准备修改语言实现，再进入[设计与贡献](/design/)，不必把路线图当作入门前置。

正式发行版见[下载中心](https://yanxu.dev/download/)，语言实现与可运行示例见[YanXuLang/yanxu](https://github.com/YanXuLang/yanxu)。
