## 安装

### macOS 与 Linux

```sh
curl -fsSL https://raw.githubusercontent.com/YanXuLang/yanxu/main/scripts/install.sh | sh
```

### Windows PowerShell

```powershell
irm https://raw.githubusercontent.com/YanXuLang/yanxu/main/scripts/install.ps1 | iex
```

安装后验证版本：

```sh
yanxu --version
```

## 第一份言序程序

新建 `初见.yx`：

```yanxu
令 名字：文 为 「言序」；
言 「你好，」 加 名字；
```

执行文卷：

```sh
yanxu 初见.yx
```

终端将输出 `你好，言序`。源码使用 UTF-8，推荐扩展名为 `.yx`。

## 命令行与 REPL

| 命令 | 用途 |
| --- | --- |
| `yanxu 文卷.yx` | 执行源码文件 |
| `yanxu` | 进入多行交互环境 |
| `yanxu --version` | 显示版本 |
| `yanxu --help` | 显示命令说明 |

REPL 会保留先前定义，函数、类、判断和循环可以跨多行输入。使用 `:助`、`:清`、`:载` 与 `:退` 管理会话。
